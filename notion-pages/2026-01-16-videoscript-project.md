## 2. Centurion Say 서비스 개요

피부과 상담 과정을 AI가 실시간으로 분석/제안하는 AI 상담 어시스턴트 서비스

### 핵심 기능

- AI 상담 녹음 및 분석: 상담 내용 실시간 텍스트 변환, 키워드 자동 추출
- 고객 이력 기반 추천: 과거 시술 이력을 바탕으로 맞춤 시술 추천
- 실시간 통역: 외국인 고객 대상 음성+텍스트 통역 제공
- AI 상담 코파일럿: 상담 중 실시간 고객 정보 가이드 제공
- 상담 평가 리포트: 상담 종료 후 점수 및 개선점 제시
- 데이터 자동 저장: 상담 결과 DB 저장 및 Admin 대시보드 연동

### 제품 구성

- PC 웹 (상담사용): 고객 목록 조회, 실시간 AI 분석, 상담 리포트 생성
- 모바일 웹 (녹음/통역용): 음성 녹음, 실시간 통역 (2분할 화면)
- PC 웹 Admin (관리자용): 고객 관리, 상담 모니터링, 대시보드

## 3. 숏폼 콘텐츠 전략

### 컨셉

"피부과에서 실제로 일어나는 웃기고 공감되는 에피소드"

### 홍보 목표

- 브랜드 인지도 향상 (제품명 직접 노출 X)
- 타겟층 공감 유도 -> 자연스러운 니즈 환기
- 바이럴 가능한 재미있는 콘텐츠 제작

### 타겟 오디언스

- 1차 타겟: 피부과/성형외과 상담사, 코디네이터
- 2차 타겟: 병원 원장, 관리자
- 3차 타겟: 피부과에 관심 있는 일반인

### 톤앤매너

- 유머: 과장된 표정, 상황의 아이러니, 예상치 못한 반전
- 공감: "맞아 맞아 나도!" 반응 유도
- 현실감: 실제 피부과에서 있을 법한 상황
- 가벼움: 무겁지 않은 일상 브이로그 느낌

## 4. 에피소드 카테고리

총 100개 에피소드를 7개 카테고리로 구성

- 고객 정보 혼동 (EP.01-15): 고객 이력 기반 추천, 데이터 관리
- 외국인 고객 통역 (EP.16-30): 실시간 다국어 통역
- 상담 기록/메모 (EP.31-45): 자동 녹음, 텍스트 변환, 자동 저장
- 시술 추천 (EP.46-60): AI 시술 추천, 고객 이력 분석
- 상담사 고충 (EP.61-75): 상담 효율성 향상, 업무 부담 감소
- 웃긴 고객 유형 (EP.76-90): 상담 효율화 (간접)
- 상담 피드백/평가 (EP.91-100): 상담 평가 리포트, 피드백 제공

### 대본 구조 (25-35초)

- 도입 0-5초: 상황 설정, 훅
- 전개 5-20초: 에피소드 진행
- 클라이맥스 20-28초: 웃음/공감 포인트
- 마무리 28-35초: 자막으로 임팩트
